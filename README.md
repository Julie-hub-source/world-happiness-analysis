# World Happiness Inequality Analysis

Statistical analysis of happiness inequality across continents using the World Happiness Report (2015–2021).

## Overview
This research examines whether happiness inequality exists across Europe, Asia, Africa,
and America — and identifies the key socioeconomic drivers behind it.

## Key Findings
- Statistically significant difference in happiness between Europe and Africa (p = 6.05E-17)
- Europe mean = 6.499 vs Africa mean = 4.727 (global average = 5.53)
- GDP per capita (r = 0.79) and social support (r = 0.76) are the strongest correlating factors

## Methods
- Two-sample hypothesis testing (t-test)
- One-sample hypothesis testing
- ANOVA + Bonferroni multiple comparison
- Correlation analysis

## Tech Stack
- Python, Pandas, Matplotlib
- Statistical testing: scipy.stats
